
# IronGuide Pro - Offline PWA Workout Tracker

This is a modern, offline-first progressive web app (PWA) designed for hypertrophy training.

## Features
- **Offline Capable:** Works without internet.
- **Local Storage:** Your data stays on your phone.
- **Active Pacer:** Voice counting (Text-to-Speech) for perfect tempo.
- **Smart History:** Remembers your weights and PRs.
- **Creatine & Water Tracker:** Detailed stats for consistency.

You do NOT need an App Store. This is a PWA.

### Option 1: Using Vercel / Netlify (Free & Best)
1. Upload this code to a GitHub repository.
2. Connect the repo to Vercel.com or Netlify.com (free account).
3. They will give you a URL (e.g., `my-gym-app.vercel.app`).
4. Open that URL on your phone.

### Option 2: Running Locally (WiFi)
1. Ensure your computer and phone are on the **same WiFi**.
2. Run `npm start` on your computer.
3. Find your computer's IP address (e.g., `192.168.1.5`).
4. On your phone, go to `http://192.168.1.5:3000`.
